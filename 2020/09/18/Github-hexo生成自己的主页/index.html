Github

  • 新建repository

    • 注意：这里的repository的名字必须要和自己Github的名字一样才可以

  • 然后里面的theme那些都可以不用选

hexo

  • 提前安装好Git和Node.js

    • Git
    • Node.js

  • 安装hexo：

建立博客

新建博客文件夹：

  • mkdir my_blog
  • hexo init my_blog
  • 可以参考 hexo官网
  • cd my_blog 进入自己的博客文件夹

常用的hexo命令

  • 写文章：hexo new [layout] <title>

    • 其中layout默认是：post，放在source/_posts下面

  • 可以在_config.yml中设置：

    • new_post_name: :year-:month-:day-:title.md，这样生成的文件名为年-月-日-title
    • post_asset_folder: true可以在_post中生成同名的文件夹，用于存放图片等东西

  • 写完文章以后，用hexo -g生成一些静态文件

  • 在用hexo -d进行部署即可
